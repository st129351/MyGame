\doxysubsection{Detailed Description}
Class representing the player character in the game. 

\doxylink{class_player}{Player} inherits from the \doxylink{class_fighter}{Fighter} base class and implements all player-\/specific functionality, including movement, combat, inventory management, and interaction with NPCs and items.
