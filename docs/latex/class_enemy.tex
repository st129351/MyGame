\doxysubsection{Detailed Description}
Base class for all enemies in the game. 

The \doxylink{class_enemy}{Enemy} class inherits from \doxylink{class_fighter}{Fighter} and adds properties and functions specific to enemies, such as enemy description, IQ, position, and experience points.
